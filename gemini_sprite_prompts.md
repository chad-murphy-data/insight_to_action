## TIPS FOR GEMINI

- If Gemini gives you a 2x2 grid instead of one image, re-run the prompt — the "Output exactly one image" instruction usually fixes it on retry.
- All prompts use a solid magenta (#FF00FF) background for easy chroma-key removal. Use "Select by Color" in GIMP, Magic Wand in Photoshop, or remove.bg to strip it out and export as transparent PNG.
- If a sprite comes back with random text/labels on it, emphasize "absolutely no text, no letters, no words anywhere in the image" more aggressively.
- For carry icons: generate at 512x512 as prompted, then downscale to 128x128 in your image editor for the sharpest result.
- If style drifts between sprites, paste 2-3 of your best results as reference images alongside the next prompt to anchor the style.
- All station sprites should be the same relative scale to each other (the "70-80% canvas height" instruction handles this). The counter is intentionally shorter at 60-70%.
